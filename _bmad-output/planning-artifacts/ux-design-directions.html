Offline Mode Direction
Offline is never hidden ‚Äî it's surfaced as a visible, calm status. Amber sync banner at top of Dashboard shows count of pending items. Each item card shows ‚è≥ Pending badge. App remains fully functional: FAB still present, scanning still works, items save as local drafts. When connectivity returns, background sync fires silently ‚Äî banner transitions to "Syncing..." then disappears with a brief ‚úì flash. No crashes. No data loss. No anxiety.

Color Palette Rationale
Dark-mode-first palette chosen for SnapLog. Purple primary (#7C6EF8) differentiates from generic Material blue while feeling intelligent and modern. Teal AI accent (#64DFDF) is used exclusively to signal AI-generated content ‚Äî creating a learnable visual language: "teal = AI did this." The background is near-black with a subtle blue undertone (#0F0F13) that feels intentional rather than default. Amber and green are exclusively used for sync state communication ‚Äî color is never the only signal; icons and text always accompany color. All text/background combinations meet WCAG AA contrast ratio (4.5:1 minimum).
